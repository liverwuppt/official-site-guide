# official-site-guide

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖不同场景下的静态页面资源。仓库本身不绑定任何特定域名或网站，仅作为页面素材的存储与版本管理用途。

## 项目简介

`official-site-guide` 是一个以 HTML 页面为主要内容的归档仓库。你可以在此找到一系列独立的、可直接运行的网页文件，适用于快速浏览、参考学习或作为静态页面部署的原始素材。

所有页面均为独立编写，彼此之间无动态依赖关系，便于单独提取使用。

## 目录说明

仓库根目录下包含多个 HTML 文件及可能的资源文件夹，大致结构如下：

```
/
├── page-1.html
├── page-2.html
├── assets/
│   ├── css/
│   ├── js/
│   └── images/
└── README.md
```

- **根目录 HTML 文件**：每个文件代表一个完整的独立页面，可直接在浏览器中打开查看。
- **`assets/` 目录**：存放页面所引用的样式、脚本及图片等静态资源，按类型分文件夹组织。

## 页面归档说明

- 所有页面均为静态 HTML，无需后端服务即可运行。
- 页面内容可能涉及多种主题或用途，每个页面在其文件头部或注释中会有简要说明。
- 如果你需要部署某个页面，请确保同时复制其引用的资源文件（如 CSS、JS、图片）。

## 维护说明

- 本仓库接受对现有页面的修正性更新，但不会频繁更改页面结构或内容核心。
- 新增页面时，请保持文件命名清晰，并在必要时更新本 README 中的目录结构示例。
- 如有资源文件更新，请同步检查所有引用该资源的页面是否正常。

## 使用建议

- 你可以直接克隆本仓库到本地，在浏览器中打开任意 HTML 文件预览。
- 若需将页面部署到服务器，建议保持文件相对路径不变，以免资源加载失败。
- 欢迎提出改进建议或报告页面显示问题，请通过 Issues 提交。
